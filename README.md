# react-redux-node-auth-sample
Example application featuring a functional Front-end React/Redux app and a node.js/express backend

## Prerequisites :
- nodejs installed

## Getting started : 

### Server
- `cd api`
- `cp .env.default .env`
- `make`

### Server tests
- `cd api`
- `make tests`

### Client
- `cd client`
- `make`
- Open http://localhost:9000 in a browser
