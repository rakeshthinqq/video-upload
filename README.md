# react-redux-node-auth-app
App- React/Redux and a node.js/express backend

## Prerequisites :
- nodejs installed

## Getting started : 

### Server
- `cd api`
- `cp .env.default .env`
- `make`

### Server tests
- `cd api`
- `make tests`

### Client
- `cd client`
- `make`
- Open http://localhost:9000 in a browser
- sign up first
   test@test.com
   test
- then sign in
